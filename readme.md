# Loop Studios Landing Page

A modern, immersive, and fully responsive landing page built as part of an interactive UI exploration series. This is my
**second standalone project** focused entirely on mastering utility-first design architectures via **Tailwind CSS v4**.

## 🚀 Features

- **Fluid Grid Layouts:** Leverages Tailwind v4's dynamic grid layouts to scale gracefully from 1-column mobile layouts
  to multi-column desktop grids.
- **Custom CSS Engine Architecture:** Configured with specific CSS-first global configurations (`@theme`, custom
  typography tokens, breakpoint mappings).
- **Interactive Component Hovers:** Elegant continuous transitions, scale transforms, and gradient overlays mapped to
  individual card containers.

## 🛠️ Built With

- **HTML5:** Semantic architecture layout structure.
- **Tailwind CSS v4.3.1:** Utilizing the next-generation native `@tailwindcss/cli` engine.
- **Google Fonts:** Integration of *Josefin Sans* and *Alata*.
